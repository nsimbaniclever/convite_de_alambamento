# 💍 Convite de Casamento - Manuel & Teresa

## 📖 Sobre o Projeto

Este é um projeto de **convite de casamento digital** criado para celebrar a união de **Manuel Oséias Nsimbani** e **Teresa Lubongo Manuel**. O projeto foi desenvolvido com muito carinho e atenção aos detalhes, combinando elementos tradicionais do alambamento (cerimônia tradicional angolana) com uma experiência digital moderna e interativa.

---

## 🎨 Estrutura do Projeto

O projeto é composto por **duas páginas interativas**:

### 📄 Página 1: Envelope Interativo (`index.html`)

Uma experiência que simula a abertura de um **envelope de convite físico**:

| Elemento | Descrição |
|----------|-----------|
| **Envelope 3D** | Design realista com abas, bordas e efeitos visuais |
| **Botão "ABRIR"** | Interação para abrir o envelope e revelar a mensagem |
| **Carta personalizada** | Mensagem de boas-vindas com ornamentos e assinatura |
| **Botão "Avançar"** | Navegação para a página principal do evento |
| **Partículas animadas** | Efeito visual de fundo com partículas flutuantes |

## 🎯 Funcionalidades Principais

### 1. 📨 Envelope Interativo
- Animação de abertura suave
- Design realista com sombras e perspectivas
- Transição fluida para a carta interna

### 2. ⏰ Contagem Regressiva
- Atualização em tempo real
- Exibição de dias, horas, minutos e segundos
- Animação visual dos números

### 3. 📝 Formulário de Confirmação
- Modal com design elegante
- Campos validados (nome, email, comparecimento, mensagem)
- Feedback visual de envio (spinner)
- Mensagem de sucesso com animação
- Envio de presente por email (simulado)

### 4. 🎨 Design Visual
- **Cores**: Tons dourados (#d4af37), branco e rosa
- **Emojis flutuantes**: ❤️💍💐✨🎉
- **Partículas animadas**: Efeito de magia no fundo
- **Fonte**: Tipografia elegante e romântica
- **Responsivo**: Adaptação para desktop, tablet e mobile
